Create a Calculator
1. Create a new Node.js project named 'Calculator'.
2. On the home page (route "/"), show a welcome message and a link to the calculator page.
3. On the "/calculator" page, display a form with two input fields and a "Sum" button.
4. When the user clicks the "Sum" button, they should be taken to the "calculator-result" page, which shows the sum of the two numbers.
  - Make sure the request go to the server.
  - Create a seperate module fot the addition function.
  - Create another module to handle incoming requests.
  On the "calculator-result" page, parse the user input, use the addition module to calculate the sum, and display the result on a new HTML page.
